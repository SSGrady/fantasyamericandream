# E009 - Onboarding Redesign

## Goal

Replace spreadsheet-style character setup with tension-forward scenario cards, life priorities, concrete job offers, and module presets that set expectations for the consequence-driven loop.

## Success Criteria

- [ ] Scenario cards show fantasy hook, tension, and difficulty rating
- [ ] Life priorities selection (wealth, stability, freedom, family, impact)
- [ ] Starting job offers with salary/benefits/commute, not sector pick alone
- [ ] Slim default life setup + optional Advanced Life Setup path
- [ ] Module presets: Guided, Realistic, Volatile, Harsh, Custom
- [ ] Onboarding flows into first chapter (E006), not generic briefing
