# E008 - Consequence Theater and Visual Design (P4)

## Goal

Make consequences **felt** through editorial presentation, semantic design tokens, card grammar, motion, and a Life Compass dashboard. Transform reports into a consequence theater.

## Success Criteria

- [ ] Semantic color CSS token system (growth, hazard, opportunity, neutral, etc.)
- [ ] Typography: editorial serif headlines + interface sans body
- [ ] Card types: briefing, opportunity, hazard, learning, stakeholder, audit
- [ ] Editorial headlines and month-by-month timeline on consequence screen
- [ ] Stakeholder voices (partner, future self, recruiter, lender, planner, manager)
- [ ] Motion animations for phase transitions (respect reduced-motion)
- [ ] Life Compass: 5 dimensions (wealth, career, health, relationships, time)
- [ ] Decision Day redesigned as command center
