# User Journey - First Run

## 1. Landing

Player sees working title, tagline, and **Choose your scenario** (card list, Fantasy President pattern).

Starter scenarios (V1):

- **Class of 2026** - age 22, bachelor's, first job, student debt, delivery habit
- **Non-Target Climb** - harder recruiting, lower initial offers, upside via certs
- **Tech Reset** - high comp, equity, elevated layoff exposure
- **Medical Road** - debt, residency income, delayed peak earnings
- **Gig Economy Survival** - no benefits, variable demand
- **Create Your Own** - dashed card; full character creator

## 2. Character Setup

Sections (card grids with modifier subtext):

- Name, age band, state/metro
- Education tier (target vs non-target), major, debt
- Career sector (tech, finance, medicine, public service, …)
- Marital status (with relationship simulation toggle)
- Starting balance sheet (checking, savings, Roth, 401k, HSA)
- Habits (delivery frequency, subscriptions, cooking skill)
- Pre-existing assets/debts

## 3. Module Toggles

Grouped toggles (Fantasy President settings pattern):

| Module | Example toggle |
|--------|----------------|
| Economy | Recessions/depressions, S&P return variability |
| Labor | Ghost jobs, mass layoff frequency |
| Life | Romance, divorce, children |
| Hazards | Weather, transit, traffic |
| Housing | House-poor events (normal vs hard %) |
| Health | ER visit frequency by difficulty |
| Gig | Ride-share fulfillment rate |
| Tax | IRS audits |
| Difficulty | Easy / Medium / Hard |
| Hints | No-hints hardcore |

## 4. First Briefing (H1 2026)

Metrics ribbon: salary, take-home, rent burden, savings rate, runway, net worth.

Narrative: starting job, lease, student loan grace period ending, open enrollment.

## 5. First Decision Day

Mix of:

- **Required:** health plan selection
- **Opportunity:** 401(k) match enrollment
- **Open:** "Describe another action…"

Optional literacy quiz offer: *The First $100K* (savings rate vs returns).

## 6. Consequence Pipeline

Processing → impact analysis → reactions (partner, future self 35, recruiter) → audit.

## 7. Dashboard

Net-worth cards, contribution progress, skill tree progress, timeline history, six-month risks.

## 8. End States

- Age 65 retirement report
- Coast FIRE voluntary exit
- Insolvency / bankruptcy
- Player-chosen end + full scoring dimensions
